CSE Projects :-
A collection of Number Theory and Cryptography algorithms implemented in C and Python for my CSE coursework, there are total of 46 programs, 34 in Python and the rest in C
 Contents
Primality Tests: Miller-Rabin, Pollard's Rho, Prime Factors.
Modular Arithmetic: Chinese Remainder Theorem (CRT), Modular Exponentiation, Inverse.
Sequences: Fibonacci, Lucas Sequence, Collatz Conjecture.
Number Properties: Amicable, Perfect, Harshad, and Automorphic numbers.
 Usage
Python: Run python filename.py
